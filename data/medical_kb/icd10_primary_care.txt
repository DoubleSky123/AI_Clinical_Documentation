COVID-19 CODING RULES
  U07.1: Use ONLY when COVID-19 is confirmed by positive test
  U07.2: REMOVED from ICD-10-CM — do NOT use for suspected/probable COVID
  Z20.828: Contact with and (suspected) exposure to other viral communicable diseases
    (use for COVID exposure without confirmed infection)
  Additional codes: Add J22 (lower resp), J12.89 (viral pneumonia), or J11.89 as needed
  Note: Z20.41 and Z20.49 are NOT valid codes for COVID exposure

CODING PRINCIPLES FOR PRIMARY CARE
  1. Code to highest documented specificity — if provider documents "type 2 diabetes with
     nephropathy," use E11.21; if only "diabetes," use E11.9
  2. Code active conditions being treated/monitored — do not code historical/resolved conditions
     unless they affect current management (use Z87.x history codes)
  3. Symptom codes (R-codes): use only when no definitive diagnosis established
  4. Z-codes for preventive encounters, screenings, and follow-up visits
  5. Secondary diagnoses: code conditions that affect management of primary diagnosis
